# Linux Keyboard Tester

Simple desktop keyboard testing tool written with Tkinter.

## Features

- Clean, easy-to-read on-screen full-size keyboard layout.
- Interactive key detection with persistent color highlighting.
- Every displayed key stays lit after the first successful press.
- Reset button to clear the test and start over.

## Run

```bash
python3 keyboard_tester.py
```

## Notes

- Built for Linux key symbols (X11/Wayland compatibility depends on desktop environment).
- Numpad, modifiers, arrows, navigation keys, and function row are included.
